# Path Zero
C++ 콘솔 기반 A* 알고리즘 시각화 프로젝트

## 1. 주요 기능 (Features)

* **직관적인 마우스 조작:** 콘솔 창임에도 불구하고 마우스 클릭과 드래그를 지원하여 간편하게 메뉴를 선택하거나 맵을 편집할 수 있습니다.
* **실시간 시뮬레이션 및 로깅:** 불길의 확산과 생존자의 이동이 실시간으로 이루어지며, 각 생존자의 상태(탈출 성공, 불길에 휩싸임 등)가 우측 로깅 창에 색상과 함께 실시간으로 기록됩니다.
* **자유로운 환경 설정:** 생존자의 이동 속도와 불길이 번지는 속도를 유저가 직접 세밀하게 조절하여 다양한 난이도와 상황을 연출할 수 있습니다.
* **생존자 직접 조작 (개입):** 시뮬레이션 도중 마우스 클릭이나 **드래그를 통해 한 명 또는 여러 명의 생존자를 선택**하고 원하는 타일로 직접 이동 명령을 내릴 수 있어 더욱 동적이고 전략적인 플레이가 가능합니다.

## 2. 시뮬레이션 진행 절차 (Simulation Flow)

### 📌 메인 화면 (Main Menu)
* 게임의 시작과 종료를 선택할 수 있는 화면입니다.
* 마우스 커서를 올려 메뉴를 하이라이트하고 클릭하여 맵 편집 모드로 진입합니다.

### 📌 맵 편집 (Map Editing)
* 시뮬레이션을 위한 무대를 직접 꾸미는 단계입니다.
* 마우스 드래그를 통해 벽을 세우고, 화재의 시작점(최대 5곳)과 탈출구(최대 3곳)를 배치할 수 있습니다.
* 우측의 속도 조절 텍스트를 마우스로 클릭해 생존자 이동 속도와 불길 확산 속도를 설정한 뒤, `Enter` 키를 눌러 시뮬레이션을 시작합니다.

### 📌 시뮬레이션 (Simulation)
* 배치된 생존자들이 지정된 탈출구를 향해 스스로 이동을 시작합니다.
* 동시에 화재가 맵 전체로 점차 번져나갑니다.
* **유저가 마우스 클릭 및 드래그를 통해 원하는 생존자(들)를 선택한 뒤, 목표 타일을 클릭하여 해당 위치로 단체 이동을 지시할 수 있습니다.**
* 모든 생존자가 무사히 탈출하거나 화재에 휩싸여 더 이상 이동할 수 있는 생존자가 없어지면 시뮬레이션이 종료되고 결과 통계가 출력됩니다.

### 📌 종료 (Exit)
* 시뮬레이션 결과창이 일정 시간 유지된 후 다시 메인 화면으로 돌아옵니다.
* 메인 화면에서 종료를 선택하여 프로그램을 완전히 종료할 수 있습니다.

## 3. 조작 방법 (Controls)

* **마우스**: 메뉴 선택, 맵 그리기 및 지우기, 시뮬레이션 속도 조절, **생존자 단일/다중(드래그) 선택 및 목표 이동 위치 지정**
* **키보드 단축키 (맵 편집 시)**
  * `W` : 벽 생성
  * `F` : 불길(시작점) 생성
  * `E` : 탈출구 생성
  * `D` : 삭제 모드
  * `R` : 선택한 영역 선택 해제
  * `Enter` : 시뮬레이션 시작
  * `ESC` : 일시 정지 및 뒤로 가기
